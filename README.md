## 🛠️ 接单操作 SOP (标准流程)
1. 在 Google Drive 复制模板，生成客户的 Google Sheet。
2. 在这个 GitHub 里新建 Folder (例如命名为 `001_Alex`)。
3. 把 `Template` 文件夹里的 `index.html` copy 到新建的 Folder 里。
4. 打开那个 `index.html`，把里面的 3 个 CSV 链接替换掉！
5. 发送给客户的最终网址是：https://[你的github名字].github.io/TravelApp/001_Alex/
